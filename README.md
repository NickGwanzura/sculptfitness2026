
# SCULPT + ASKANA Wellness

A premium, inclusive wellness platform and high-end clothing label designed for the modern woman. Based in the heart of Harare, Zimbabwe, SCULPT focuses on strength, balance, and longevity through every season of life—from professional peak to the sacred transitions of motherhood.

## 🌿 The Vision

SCULPT is a curated space that merges African elegance with modern longevity science. We redefine strength not as a destination, but as the rhythm of a life well-lived.

### Key Pillars
- **Strength Training:** Foundational, functional, and future-proof movement.
- **Classical Pilates:** Precision and core alignment for a resilient physique.
- **Postnatal Wellness:** Specialized recovery honoring the journey of motherhood.
- **Nutrition:** Heritage-conscious, sustainable meal planning without the noise.
- **ASKANA Atelier:** Minimalist, high-performance studio wear designed with intention.

## 🛠 Tech Stack

This application is built as a high-performance **Single Page Application (SPA)** to ensure a cinematic and seamless user experience.

- **Framework:** [React 19](https://react.dev/)
- **Routing:** [React Router 7](https://reactrouter.com/)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Typography:** Playfair Display (Serif) & Inter (Sans-serif)
- **Deployment-Ready:** Configured with `HashRouter` for zero-config hosting compatibility.

## 🚀 Getting Started

### Prerequisites
- A modern web browser.
- A local development server (e.g., VS Code Live Server, Vite, or any static file server).

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/sculpt-wellness.git
   ```
2. Open `index.html` in your browser via a local server to handle the ES6 modules and React mount.
